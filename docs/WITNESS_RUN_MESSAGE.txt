NEURALOGIX VOR WITNESS RUN — v0.7.3-public
==========================================

This message contains instructions to verify the NeuraLogix Verified Observation Runtime
on a clean machine. The goal is to confirm that the public demo package produces
deterministic, hallucination-free outputs under the audited conditions.

PREREQUISITES
-------------
- Python 3.9+ (verified on 3.11)
- Git
- ~500MB disk space

VERIFICATION COMMANDS (copy-paste block)
-----------------------------------------

### Windows
```powershell
git clone https://github.com/CULPRITCHAOS/VOR.git
cd VOR
git checkout v0.7.3-public
python -m venv .venv
.\.venv\Scripts\activate
pip install -e .[dev]   # REQUIRED: must use [dev] extras for test dependencies

pytest
neuralogix pack validate --pack data/packs/public_demo_v0_7_1
neuralogix pack validate --pack data/packs/adversarial_v1
neuralogix audit --fast
```

### Linux/macOS
```bash
git clone https://github.com/CULPRITCHAOS/VOR.git
cd VOR
git checkout v0.7.3-public
python3 -m venv .venv
source .venv/bin/activate

pytest

neuralogix pack validate --pack data/packs/adversarial_v1

```

WHAT THIS PROVES
----------------
A successful witness run demonstrates:
- The VOR kernel correctly rejects ungrounded observations
- The Truth Gate produces 0% hallucination on audited packs
- Evidence trails are cryptographically reproducible
- The system abstains when evidence is insufficient (no fabrication)
- All verification gates pass (parity, integrity, calibration)

WHAT THIS DOES NOT INCLUDE
--------------------------
This public demo excludes:
- Proprietary corpus ingestion pipelines
- Internal/customer-specific evaluation packs
- Pack generation recipes and gold-label construction
- Domain-specific parsing extensions
